# Customer Call Centre Data Analysis (Power BI)

## Overview
A Power BI dashboard that analyzes call centre performance with insights into call volume, agent efficiency, customer satisfaction, and response speed.

## Key Insights
- Total calls: 7K+
- Answered, rejected, resolved & unresolved call breakdown
- Calls by topic (Streaming, Technical Support, Payments, etc.)
- Agent-wise performance & satisfaction rating
- Average speed of answering calls
- Calls answered vs rejected by topic and agent


## Insights Generated
- Streaming & Technical Support categories receive the most calls.
- Jim, Becky, and Greg show strong resolution rates.
- Slightly lower satisfaction scores highlight training opportunities for specific agents.

- Make faster, data-driven decisions
